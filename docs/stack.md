# Стек проекта

## Основные технологии

- **React** - библиотека для построения пользовательских интерфейсов
- **React Router** - маршрутизация в React-приложениях
- **Redux** - управление глобальным состоянием приложения
- **Redux Persist** - сохранение состояния Redux в локальное хранилище
- **Material UI (MUI)** - библиотека компонентов пользовательского интерфейса на основе Material Design

## Инструменты разработки

- **Vite** - инструмент сборки и dev-сервер
- **TypeScript** - типизированный надмножество JavaScript
- **Vitest** - фреймворк для тестирования
- **ESLint** - линтер для проверки качества кода
- **Yarn** - пакетный менеджер
